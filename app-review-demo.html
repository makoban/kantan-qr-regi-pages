Kantan QR Regi Demo QR

This page is for App Review. It provides a sample product data file and the matching QR code for testing the QR scanning flow.

Test steps

  1. Download the sample product data JSON below to the review device.
  2. Open the app, then go to Settings > Import product data.
  3. Select the JSON file. The product "Demo Cookie" will be added.
  4. Display the QR code on another screen, then scan it from the register screen in the app.
  5. The app adds Demo Cookie to the cart. Complete checkout to review the receipt and sales history.
